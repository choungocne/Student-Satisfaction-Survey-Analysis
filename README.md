# Student Satisfaction Survey Analysis

## Project Overview

This project analyzes student satisfaction survey data to identify key factors affecting student experience and provide actionable recommendations.

## Objectives

* Clean and transform survey data using SQL
* Analyze satisfaction trends by gender and academic year
* Identify areas for improvement
* Build dashboards and reports for decision-making

## Tools Used

* MySQL
* SQL
* Microsoft Excel
* GitHub

## Data Cleaning

Performed:

* Missing value checking
* Duplicate detection
* Data standardization

## Key Analysis

### Average Satisfaction Score

Calculated overall satisfaction level across all respondents.

### Satisfaction by Gender

Compared satisfaction levels between male and female students.

### Satisfaction by Academic Year

Identified differences among Year 1–4 students.

### Service Evaluation

Analyzed:

* Facilities
* Teaching Quality
* Student Support

## Key Findings

* Senior students reported higher satisfaction scores.
* Facilities received lower ratings than teaching quality.
* First-year students require additional support.

## Recommendations

* Improve campus facilities.
* Strengthen onboarding programs for new students.
* Increase student support activities.
